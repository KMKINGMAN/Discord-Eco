# KINGMAN DEV

[![](https://img.shields.io/discord/565048515357835264.svg?logo=discord&colorB=blue&label=KINGMAN_DEV)](https://discord.gg/KD2mdRReSA)
[![](https://img.shields.io/badge/discord.js-v13.1.0-blue.svg?logo=npm)](https://github.com/discordjs)

Discord Eco is a open source Discord bot coded in JavaScript with [Discord.js](https://discord.js.org) and [Mongoose](https://mongoosejs.com/docs/api.html) by [KINGMAN](https://github.com/KMKINGMAN).  
Feel free to add a star ‚≠ê to the repository to promote the project!

### Many commands

Discord Eco has a lot of features

- [x] Supports slash command
- [x] new Events `DailyReseve` - `CreditSent`
- [x] Logs for `Reseve` - `Transfare` Credits
- [x] MongoDB Database

	<summary>Click Here</summary>
     <details>
     <summary>1 - Make Discord Bot</summary>

		<summary>2 - install packages</summary>

	     </details>
	    <details>
		<summary>2- add token - mongodb connection to .env</summary>
        <code>
        KM_MONGO=
        TOKEN=
        </code>
	</details>
	<details>
		<summary>Edit Config File `LocalDB/config.json`</summary>
		<code>
		{
            "prefix": "#"
        }
		</code>
	</details>
	<details>
		<summary>Run The Project</summary>
		<code>
		npm start
		</code>
	</details>
</details>

* if you use Repl.it You Need To Update NodeJs to v16 

```shell
npm init -y && npm i --save-dev node@16 && npm config set prefix=$(pwd)/node_modules/node && export PATH=$(pwd)/node_modules/node/bin:$PATH
```
